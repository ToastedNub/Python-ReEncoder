# Python-ReEncoder
 - ReEncodes Videos with Various Encoders

Requires FFMPEG

Install: https://ffmpeg.org/download.html
Install Guide: https://youtu.be/JR36oH35Fgg




Put your Videos in the MEDIA folder

Run the "Start" file

Your export will be in the EXPORT folder





ENCODERS

This is just to show what all can view each encoded file


H.264/libx264 - All Devices
HEVC/H.265/libx265 - All MODERN Devices
AV1 - 40 Series GPUs and higher only
VP9 - WebM container
VP8 - Weaker VP9
ProRes - MacOS
MJPEG - Most Devices (legacy support encoder)
FFV1 - Purely archival, not for sharing
GIF - All Devices
FRAMES - All Devices

BEST FOR EVERYONE
H.264, or lib264

HEVC, or libx265 is 264 is too slow
